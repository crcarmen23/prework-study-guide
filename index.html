HTML

  • Head element: contains information about the webpage.
  • Body element: represents the visible content shown to the user.
  • Meta element: contains information about the page that is used by the browser.
  • Title element: holds the title for the webpage.
  • Child elements: elements nested inside other elements. The parent element holds all of the children.
  • Semantic elements: elements in which the name of the element describes its content.

CSS

  • Margin: indicates how much space we want around the outside of an element
  • Padding: indicates how much space we want around the content inside an element
  • Selector: defines the element/attributes to which the rules will apply
  • External CSS Stylesheet: most common way to style a webpage, allows you to keep all CSS rules in a separate file which makes design changes easier

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches to a specific branch
  • git add -A: stage your changes, commits all files in the working branch
  • git commit -m "your message": commits the changes and adds a short descriptive message of what changes were implemented
  • git pull origin main: pull in the base branch, do this before pushing up any code
  • base branch is what we want to merge into, compare branch is the branch we want to merge

Javascript

  • Variable: named container that allows us to store data in our code
  • Control flow: the order in which a computer executes code in a script
  • Each line of Javascript ends with a semi-colon
  • If: a conditional statement that introduces decision making into the code
  • Array: single variable used to hold a group of data that is usually related in some way
  • Index: unique number used to identify the location of a data item in an array
  • For loop: uses the predictable pattern of indices to perform a task on all items in an array by allowing a single code block to be executed over and over
  • Function: set of instructions that tells the computer how to perform a certain task
